## ✨ Features

- ✅ **Premium Animated Hero** with typing effect
- ✅ **Sticky Navbar** with mobile hamburger menu
- ✅ **Scroll Progress Bar**
- ✅ **Counter Animation** (statistics)
- ✅ **Scroll Reveal** (intersection observer)
- ✅ **Glassmorphism UI** with gradients
- ✅ **Floating Instagram Button**
- ✅ **Back to Top Button**
- ✅ **FAQ Accordion** (expandable)
- ✅ **Contact Form** with feedback
- ✅ **Newsletter Subscription** (UI only)
- ✅ **Page Loader** with spinner
- ✅ **Lazy Loading** for images
- ✅ **SEO Ready** (meta tags, OG, Twitter Cards)
- ✅ **Accessibility** (ARIA labels, keyboard navigation)
- ✅ **Mobile First** responsive design
- ✅ **Modern Animations** (hover, transitions)
- ✅ **No external frameworks** — pure HTML/CSS/JS

## 🛠️ Tech Stack

- **HTML5** — semantic markup
- **CSS3** — custom properties, flexbox, grid, animations
- **JavaScript (ES6)** — DOM manipulation, Intersection Observer, event handling
- **Google Fonts** — Inter
- **Font Awesome** — icons
- **Unsplash** — royalty-free placeholder images
